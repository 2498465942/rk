# 数据转换器
纯前端本地运行的数据格式转换工具，一键将单列数据按空行分组并转为逗号分隔格式。

## 功能特性
- 单列数据按空行自动分组
- 一键转换 / 复制 / 清空
- 支持 Ctrl+Enter 快捷键转换
- 结果自动复制到剪贴板
- 响应式布局，支持电脑/手机
- 本地处理数据，无需联网

## 项目结构
├── 数据换器.html  # 主工具文件（双击直接使用）
└── README.md     # 说明文档

## 使用方法
1. 双击打开 数据换器.html
2. 左侧输入框粘贴数据（空行分组）
3. 点击【一键转换】或 Ctrl+Enter
4. 右侧输出结果并自动复制

## 数据示例
输入：
编码1
编码2

编码3
编码4

输出：
编码1,编码2
编码3,编码4
